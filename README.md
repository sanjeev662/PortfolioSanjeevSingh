## 🚀 Getting Started

### Prerequisites
- Node.js (v14+) and npm

### Installation
```bash
git clone https://github.com/sanjeev662/PortfolioSanjeevSingh.git
cd PortfolioSanjeevSingh
npm install
npm start
```

Navigate to `http://localhost:3000` to view the portfolio.

### Available Scripts
- `npm start` - Development mode
- `npm build` - Production build
- `npm test` - Run tests
